Schedule Planner

Directions:
  • Enter each subject and set its priority (1=Low, 5=High).
  • Set difficulty (1=Easy, 5=Hard) and proficiency (1=Weak, 5=Strong).
  • Optionally, add exam date (subjects with sooner exams get more hours).
  • Set min/max hours if you want to limit time for a subject.
  • Default values are chosen if you leave a field blank.
Example: Math (priority 5, difficulty 4, proficiency 2, exam date 2025-10-10, min 2, max 6)
